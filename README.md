# Muhyideen Muhammad | Frontend Developer Portfolio

A high-end, premium portfolio showcasing specialized frontend expertise in building performance-optimized, scalable, and visually stunning web applications.

## 🚀 Overview

This portfolio is built with a focus on modern web aesthetics, smooth user experience, and technical excellence. It features a glassmorphic design system, dynamic animations, and a fully responsive layout tailored for all devices.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Fonts**: Inter & Outfit (via Google Fonts)

## ✨ Key Features

- **Glassmorphic UI**: Ultra-modern design using backdrop blurs and subtle borders.
- **Dynamic Projects Showcase**: Real-world project cards with high-quality visuals and hover effects.
- **Interactive Timeline**: A clean, professional representation of career growth and experience.
- **Performance Optimized**: Fast load times and smooth scrolling implemented via standard CSS and lightweight libraries.
- **Custom Branding**: Bespoke SVG favicon and custom color tokens integrated into Tailwind.

## 📁 Portfolio Structure

```bash
├── public/                # Static assets & project images
├── src/
│   ├── components/        # Reusable UI components (Hero, Navbar, etc.)
│   ├── data/              # Centralized portfolio data (portfolio.ts)
│   ├── App.tsx            # Main application entry
│   └── index.css          # Global styles & Tailwind layers
└── tailwind.config.js     # Custom design system configuration
```
